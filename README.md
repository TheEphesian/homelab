# Homelab

Android SSH application with Tailscale integration for secure remote access to homelab devices.

## Description

This project is an Android SSH client that integrates with Tailscale to provide secure, encrypted remote access to your homelab devices. By leveraging Tailscale's WireGuard-based VPN, you can securely connect to your home network services without exposing them to the public internet.

## Features

- SSH client functionality for Android
- Tailscale integration for secure network access
- Seamless connection to homelab devices
- Encrypted communication
