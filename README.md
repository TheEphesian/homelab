# Homelab

Android SSH application with Tailscale integration for secure remote access to homelab devices.

## Features

- Secure SSH connections to homelab devices
- Tailscale integration for zero-config VPN access
- Android-native UI
- Kotlin-based implementation

## Getting Started

### Prerequisites

- Android Studio Arctic Fox or later
- JDK 11 or later
- Android SDK 21+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TheEphesian/homelab.git
   ```

2. Open in Android Studio
3. Build and run on your Android device

## Usage

1. Launch the app
2. Add your homelab devices
3. Connect via SSH through Tailscale
